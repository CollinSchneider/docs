**Swishjam supports multiple API keys:**
- Segment & combine data easier
- Pre-built dashboards

<Accordion title="Learn About Web vs Product Analytics API Key">
    Swishjam supports multiple API keys to help keep you data separated but easily connectable. We currently create 2 distinct API keys for you web and product keys. Below is more about each 
    <CardGroup cols={2}>
        <Card title="Web Analytics API Key" icon="filter">
            The web analytics API key is meant for your landing pages, docs, blog, and other similar sites. The Web Analytics dashboard will auto-populate with data regarding page views, browser types, etc on users that land on these pages.
        </Card>
        <Card title="Product Analytics API Key" icon="arrow-pointer">
            The product API key is meant for your app. Since you've sparated web from product we're able to easily pre-build a product usage dashboard for your users.
        </Card>
    </CardGroup>

    Both keys feature the same features in terms of tracking events and identifying users. The core difference is in utilizing the pre-build dashboards out of the box.
</Accordion>
