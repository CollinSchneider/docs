### 2. Update with your API key
Update the API key in the javascript snippet above (`swishjam-XXXXXXXX`) with your API from the <a target="_blank" href="https://app.swishjam.com/settings">Swishjam settings page</a>.  We have multiple API keys to separate data from your various sites & apps. We automatically create 2 keys for "marketing" and "product" for you to use.

### 4. Identifying Users

If your users login or completes a form and you would like to identify their activity in Swishjam you can use the .indentify function.
<Steps>
    <Step title="Verify Swishjam is loaded in your footer">
    </Step>
    <Step title="Trigger your event">
        ```javascript
        swishjam.identify('user@example.com', {
            ...include your data here
        })
        ```
    </Step>
</Steps>
